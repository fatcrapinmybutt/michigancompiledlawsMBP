## 📚 Overview

This repository contains a **complete HTML archive** of the Michigan Compiled Laws (MCL), providing primary source material for all Michigan statutes. With **111MB** of content across **100+ HTML files**, this is the most comprehensive collection of Michigan statutory law available.

## 🎯 Key Chapters

### Constitution & Government
- **Chapter 1** - Constitution of Michigan 1963 (174KB)
- **Chapter 2-4** - Elections, Form of Government, Legislature
- **Chapter 15-17** - State Agencies, Executive Organization, Administrative Procedures

### Judiciary & Courts
- **Chapter 600** - Revised Judicature Act (5.4MB) - *Core judicial authority*
- **Chapter 600-8301** - District Court Civil Jurisdiction ($25,000 limit)
- **Chapter 600-8401** - Small Claims Jurisdiction ($7,000 effective 2024)

### Family Law
- **Chapter 722** - Children (1.7MB) - *Child Custody Act, Parenting Time, Support*
- **Chapter 712A** - Juvenile Code
- **Chapter 710** - Probate Code

### Criminal Law
- **Chapter 750** - Penal Code (3.2MB) - *All criminal offenses*
- **Chapter 760-777** - Code of Criminal Procedure (1.3MB+)
- **Chapter 780** - Corrections

### Civil & Commercial
- **Chapter 440** - Uniform Commercial Code
- **Chapter 450** - Business Organizations
- **Chapter 554** - Real and Personal Property
- **Chapter 560** - Conveyances
- **Chapter 565-570** - Landlord-Tenant, Housing

### Special Topics
- **Chapter 37** - Elliott-Larsen Civil Rights Act
- **Chapter 15** - Whistleblower Protection
- **Chapter 333** - Public Health Code
- **Chapter 257** - Michigan Vehicle Code
- **Chapter 418** - Workers' Compensation
- **Chapter 500** - Insurance Code
